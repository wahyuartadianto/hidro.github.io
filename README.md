Kalkulator & Panduan Hidroponik 🌱

Rencanakan sistem hidroponik Anda dari awal hingga panen dengan mudah dan akurat.

### Fitur Utama
- Kalkulator nutrisi dan kebutuhan media tanam
- Simulasi pemilihan tanaman hidroponik
- Estimasi biaya dan jadwal panen
- Panduan lengkap lengkap dengan analisa hasil
- Ekspor hasil ke PDF & Excel
